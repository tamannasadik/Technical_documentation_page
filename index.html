Introduction to React

React is a JavaScript library for building user interfaces.

It lets you compose complex UIs from small and isolated pieces of code called "components."

React was created by Jordan Walke, a software engineer at Facebook.

const element = <h1>Hello, world!</h1>;
  • Declarative
  • Component-Based
  • Learn Once, Write Anywhere
JSX in React

JSX stands for JavaScript XML.

It allows you to write HTML inside JavaScript and place them in the DOM without any createElement() and/or appendChild() methods.

const element = <h1>Hello, world!</h1>;

JSX makes your React code simpler and elegant.

You can embed any JavaScript expression in JSX by wrapping it in curly braces {}.

Components and Props

Components let you split the UI into independent, reusable pieces.

A function component is a simple way to define a component.

function Welcome(props) { return <h1>Hello, {props.name}</h1>; }

Props are read-only.

  • Function Components
  • Class Components
State and Lifecycle

State is similar to props, but it is private and fully controlled by the component.

Components can manage their own state using the `useState` hook.

const [count, setCount] = useState(0);

The component lifecycle contains several methods like `componentDidMount`, `componentDidUpdate`, and `componentWillUnmount`.

Handling Events

Handling events with React elements is very similar to handling events on DOM elements.

You can pass an event handler function as a prop to a React component.

function handleClick() { alert('Button clicked!'); }

To handle an event in React, you use the camelCase syntax, like `onClick`.
